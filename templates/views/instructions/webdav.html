How to connect WebDAV clients to Webtor

You can access your personal torrent library from Webtor through WebDAV protocol. This allows you to manage torrents and stream content using standard file managers and WebDAV clients.

üìÅ Folder Structure

  • torrents/ - Manage your torrent library (add/remove .torrent files)
  • all/ - Browse all your content (read-only)
  • movies/ - Browse movies only (read-only)
  • series/ - Browse TV series only (read-only)

üñ•Ô∏è Windows Explorer

  1. Open File Explorer
  2. Right-click on This PC and select Map network drive...
  3. Choose a drive letter and enter your WebDAV URL in the folder field
  4. Check Connect using different credentials if prompted
  5. Click Finish and your WebDAV folder will appear as a network drive

üçé macOS Finder

  1. Open Finder
  2. Press Cmd+K or go to Go ‚Üí Connect to Server
  3. Enter your WebDAV URL in the server address field
  4. Click Connect
  5. The WebDAV folder will appear in the sidebar under Shared

üêß Linux (using rclone)

# Install rclone if not already installed
sudo apt install rclone  # Ubuntu/Debian
# or
brew install rclone      # macOS with Homebrew

# Configure WebDAV remote
rclone config

# Choose: n) New remote
# Name: webtor
# Storage: webdav
# URL: [Your WebDAV URL]
# Vendor: other
# User: [leave empty]
# Password: [leave empty]

# Mount the WebDAV folder
mkdir ~/webtor
rclone mount webtor: ~/webtor --daemon

# Or sync files
rclone sync webtor:torrents ~/local-torrents

üîß Troubleshooting

Connection refused or timeout errors

Make sure you're using the correct WebDAV URL from your profile page. Check your internet connection and firewall settings.

Authentication errors

WebDAV uses token-based authentication embedded in the URL. Make sure you're using the complete URL from your profile page without modifications.

Files not appearing or slow loading

Large libraries may take time to load. Try refreshing the folder or accessing specific subfolders. Some WebDAV clients cache directory listings.
