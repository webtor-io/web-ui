Feature list

Direct Download Link (DDL) - select any file inside torrent and download it without VPN and torrent-client.

Instant video and audio streaming - select video or audio file inside torrent and start watching it without need of download. Supported formats:

  • Video: avi, mkv, mp4, webm, m4v, ts, vob
  • Audio: mp3, wav, ogg, flac, m4a

Download whole torrent as ZIP-archive - download your torrent as ZIP-archive on-the-fly preserving original directory structure without torrent-client.

Personal Library – organize your own collection by adding torrents to your account. Movies and series will be detected automatically!

Stremio integration – just install the addon using the link from your profile and start watching your library on TV with Stremio.

WebDAV support – access and manage your library as a network folder using a personal link from your profile. Tested and works well with Mountain Duck, RaiDrive, Owlfiles.

Developer friendly - with the SDK you can provide your users with the ability to watch torrent-videos online on your website.

FAQ

Do I need to turn on my VPN when using Webtor.io? Will my ISP see what I'm downloading?

For an ISP, using Webtor.io is no different from using any other site such as YouTube or Google. All information between the user and Webtor.io is transmitted only over the secure HTTPS protocol, which excludes the possibility of interception. The user's IP address is not broadcasted to the BitTorrent network, since downloads from the BitTorrent network are only handled by Webtor.io servers. Thus, the use of Webtor.io is anonymous and safe without any additional measures.

Can I remove ads when using Webtor.io?

In order to remove ads, you just need to start helping the project. In addition to removing ads, you will also be able to get faster loading speed and a heart icon in the interface!

Can I use Webtor.io on my mobile phone or tablet?

Yes! You can use Webtor.io to download and stream torrents on any mobile device, without having to install additional apps.

Is Webtor.io open-source project?

All components of Webtor.io are absolutely open-source and distributed under MIT license. You can visit github repository to view source codes.

Can I watch my library on a smart TV?

Yes! With the new Stremio integration, you can stream your personal library directly on your smart TV or any device running Stremio. Just follow the addon link in your profile.

Do links or streams expire?

Yes, for security reasons, direct links and streaming sessions are temporary and may expire after a short time. You can always refresh or regenerate them from your library interface.

Why do some downloaded ZIP archives show a CRC error?

ZIP archives are generated on-the-fly and don't include full checksum data. Some extraction tools may show a warning or error even when the file is fully intact. We recommend using 7-Zip for best compatibility.

Why is my download or streaming speed low?

Download and streaming speeds may vary due to several factors:

  • The number of seeders available for the torrent
  • File format – some formats require real-time transcoding
  • Your internet connection speed and stability
  • Current load on Webtor servers
  • Your account type – premium users get higher bandwidth

Important: bandwidth is measured in Mbps (Megabits per second), while file sizes are shown in MB (Megabytes). 1 MB = 8 Mb, so a 50 Mbps connection gives about 6.25 MB/s download speed.

We do our best to optimize delivery, but performance can still differ between torrents. Supporting the project helps unlock better speed and reliability.
